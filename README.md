# WW-CommHub
一款专为新闻传播学考研人设计的纯前端学习管理工具，集成时间轴规划、专题练习、模考系统、热点聚合、文章库、文件管理与待办日历，所有数据私有存储，无需后端，开箱即用。

[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](http://makeapullrequest.com)

## ✨ 功能特性

- 📅 **备考时间轴 + 待办日历**  
  点击四个备考阶段卡片，打开可翻页的日历，支持按日期添加/删除待办事项（标题、描述、时间），每个阶段独立存储。

- 📚 **专题练习树**  
  覆盖热门新技术、网络视听、媒体融合、马新观、国际传播、媒介伦理等十大专题，点击显示核心笔记。支持**富文本编辑**（粗体、斜体、下划线、字号、颜色、格式刷等），笔记自动保存到本地。

- 🎯 **三次模考系统**  
  预设10月基础检测、11月热点检测、12月全真模拟，每套试卷包含“4名词解释+6简答+2论述”。支持**新增/删除/编辑模考卷**，所有修改自动保存，可自评分数。

- 🔥 **实时热点榜单**  
  集成公共热搜 API（微博、抖音、小红书），自动刷新，展示最新案例，无需额外配置。

- 📖 **文章库 & 文件管理**  
  - 以“标题+链接”形式收藏外部文章（知乎、公众号等），点击跳转原文，完全合规。  
  - 支持上传 PDF、Word、图片等文件，在线预览（图片/PDF）与下载。  
  - 所有文章和文件按**用户名隔离**，切换用户后数据完全独立。  
  - 内置6篇高质量新传考研文章作为初始数据。

- 👤 **用户空间隔离**  
  基于 `localStorage`，每个用户可设置独立昵称，所有笔记、待办、文章库、模考成绩、上传文件均私有存储，切换用户数据完全隔离。

- ✏️ **专题栏目自定义**  
  支持添加/删除专题分类和子专题，构建自己的知识体系。

- 📁 **资料管理**  
  可上传任意文件（图片、PDF、Word、Excel等），在本地预览或下载，方便整理学习资料。

📝 使用指南
首次打开：输入你的昵称（或使用默认用户），之后所有学习数据都会关联此昵称。

管理待办：点击时间轴任一阶段卡片 → 选择日期 → 添加/删除待办事项（支持标题、描述、时间）。

查看专题：左侧专题树点击任意子项 → 右侧显示核心笔记，点击“编辑”按钮可进入富文本编辑模式（支持格式刷、颜色、字号等）。

参加模考：点击上方模考选项卡 → 查看试卷 → 点击“开始考试”自评分数 → 成绩自动保存。点击“编辑模考”按钮可新增/删除/修改试卷。

收藏文章：在文章库区域输入标题和链接，或上传本地文件，点击“添加到文章库”即可。预设的6篇文章可作为参考。

上传资料：在“资料管理”区域可上传任意文件（支持多选），文件会保存在当前用户的本地存储中，支持预览（图片/PDF）和下载。

切换用户：在文章库区域修改用户名并点击“切换用户”，即可得到全新独立空间。
